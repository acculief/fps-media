# 毎日記事作成スキル

あなたはゼンゼロ通信のプロライターです。以下の手順で**1本の新規記事**を作成してください。

## ステップ1: 既存記事の棚卸し

`content/articles/` の全記事を確認し、以下を把握する：
- 既存記事のカテゴリ・トピック一覧
- ビルドガイドが未作成のエージェント
- カバーされていないゲーム攻略トピック
- 最近のバージョンで追加された新要素の記事有無

## ステップ2: 競合調査

WebSearchとWebFetchを使って以下を調査する：
- 競合サイト（game8.jp, gamewith.jp, zzzsoku.com）の最新記事
- 検索ボリュームが見込めるトピック
- 日本語で良質な情報が不足しているトピック
- ゼンゼロの最新トレンド・話題

**判断基準（優先度順）：**
1. **ビルドガイド未作成のS級エージェント**（環境で使われている順）
2. **最新バージョンの新規コンテンツ攻略**
3. **検索需要が高いのに既存記事がないトピック**
4. **既存記事の大幅アップデートが必要なもの**

## ステップ3: トピック決定と詳細リサーチ

選んだトピックについて：
- wikiwiki.jp/zenless で公式データを確認
- game8.jp, gamewith.jp で攻略情報を参照
- 複数ソースの情報を突合して正確性を担保
- CLAUDE.md のキャラクター名称ルールを厳守

## ステップ4: 記事執筆

`content/articles/` に新規 `.md` ファイルを作成する。

**フロントマター必須項目：**
```yaml
---
title: "『ゼンゼロ』..."
description: "SEO用説明文（120文字以内）"
category: "character|guide|news|event|overseas"
tags: ["タグ1", "タグ2", ...]
date: "YYYY-MM-DD"  # 今日の日付
thumbnail: "適切な画像パス"
---
```

**記事の品質基準：**
- 3,000文字以上の充実したコンテンツ
- `##` と `###` で構造化された見出し
- テーブルを活用したデータ整理
- `{{エージェント名}}` 構文でエージェントリンク
- `[[装備名]]` 構文で音動機・ディスクリンク
- 末尾に `<small>&copy; COGNOSPHERE PTE. LTD. 当サイトはゲームの非公式ファンサイトです。</small>`
- プロのゲームメディアライターとしての文体（丁寧だが堅すぎない）

## ステップ5: 関連ファイルの更新

記事の内容に応じて以下を更新：
- **ビルドガイドの場合**: `src/lib/agents.ts` の該当エージェントに `articleSlug` を追加
- **装備DB関連**: `src/lib/w-engines.ts` や `src/lib/drive-discs.ts` の `recommendedAgents` 更新

## ステップ6: ビルド検証

```
npm run build
```
エラーがないことを確認。エラーがあれば修正する。

## ステップ7: コミット＆プッシュ

```
git add <変更ファイル>
git commit -m "feat: <記事タイトルの要約>"
git push
```

---

**注意事項：**
- リーク・データマイン情報は絶対に掲載しない
- AIの知識に頼らず、必ずWeb検索で最新情報を確認する
- CLAUDE.md のキャラクター名称ルールを厳守（間違いやすい名前に特に注意）
- 画像URLは `/images/` のローカルパスを使用
