# ビルドガイド特化スキル

あなたはゼンゼロ通信のプロライターです。**指定されたエージェントのビルドガイド記事**を作成してください。

引数: $ARGUMENTS（エージェント名。例: "儀玄", "イヴリン"）

## ステップ1: 対象エージェントの確認

- `src/lib/agents.ts` でエージェント名が存在するか確認
- 既に `articleSlug` が設定されていないか（既存ガイドの重複防止）
- CLAUDE.md のキャラクター名称ルールで正式名称を確認

## ステップ2: 詳細リサーチ

WebSearchとWebFetchで以下を調査：

1. **wikiwiki.jp/zenless** で公式データ確認：
   - 属性、特性、所属、武器タイプ、実装バージョン
   - 全スキルの詳細（通常攻撃、特殊スキル、連携スキル、終結スキル、コアパッシブ）
   - 凸効果（心象映画）の全段階

2. **game8.jp** でビルド情報確認：
   - おすすめ音動機（モチーフ＋代替3〜4種）
   - おすすめドライバディスク（メインセット＋代替セット）
   - メインステータス・サブステータス優先度
   - おすすめチーム編成

3. **gamewith.jp** で補完情報：
   - 凸効果の評価・優先度
   - 立ち回りのコツ
   - 引くべきかの総合評価

## ステップ3: 記事執筆

ファイル名: `content/articles/zzz-{英語名}-build-guide.md`

**必須セクション構成：**

```
1. 導入文（キャラの概要と魅力を2〜3段落で）
2. ## 基本情報（テーブル）
3. ## スキル詳細（独自メカニクスを重点解説）
4. ## おすすめ音動機
   - Tier 1: モチーフ武器
   - Tier 2: 代替音動機テーブル
5. ## おすすめドライバディスク
   - 最優先セット
   - 代替セット
   - メインステータス優先度テーブル
   - サブステータス優先度
6. ## おすすめチーム編成（3〜4パターン）
   - 各チームにテーブル＋解説
7. ## 凸効果（心象映画）の評価
   - 0凸〜6凸を個別評価
   - 凸優先度まとめテーブル
8. ## 立ち回り・ローテーションのコツ
9. ## 引くべき？ 総合評価
10. 著作権表記
```

**構文ルール：**
- 音動機名は `[[名前]]` で囲む（W_ENGINE_NAME_MAP に存在する名前のみ）
- ディスク名は `[[名前]]` で囲む（DRIVE_DISC_NAME_MAP に存在する名前のみ）
- エージェント名は `{{名前}}` で囲む（AGENT_MAP に存在する名前のみ）
- 存在しない名前はプレーンテキストにする

## ステップ4: 関連ファイル更新

1. `src/lib/agents.ts`: 該当エージェントに `articleSlug: "zzz-{英語名}-build-guide"` を追加
2. `src/lib/drive-discs.ts`: 該当ディスクの `recommendedAgents` にエージェント名を追加（該当する場合）
3. `src/lib/w-engines.ts`: 該当音動機の `recommendedAgents` にエージェント名を追加（該当する場合）

## ステップ5: ビルド検証＆デプロイ

```
npm run build  # エラーなし確認
git add <変更ファイル>
git commit -m "feat: {エージェント名}ビルドガイド記事追加"
git push
```
