『VALORANT』のランクマッチでは、**味方に敵の位置を正確に伝える**ことが勝利の鍵を握る。しかし、「あそこに敵がいる」「右から来た」といった曖昧な報告では、味方は的確な判断ができない。

本記事では、VALORANTの現行マッププール全マップの主要なコールアウト（位置の呼び方）をまとめる。これを覚えるだけで、VC（ボイスチャット）での報告精度が格段に上がるはずだ。

---

## コールアウトの基本ルール

VALORANTのコールアウトには、いくつかの共通ルールがある。

- **CT**: 防衛側のリスポーン（攻撃側から見て奥側）
- **T**: 攻撃側のリスポーン（攻撃側のスタート地点側）
- **ヘブン**: サイト上の高い位置
- **ヘル**: サイト下の低い位置
- **ロング**: サイトへの長い直線通路
- **ショート**: サイトへの短い通路
- **ミッド**: マップ中央のエリア
- **メイン**: 各サイトへの主要な進入路

---

## バインド（Bind）

バインドはテレポーターが特徴的な2サイトマップ。ミッドが存在しない独特な構造を持つ。

### Aサイト周辺
| コールアウト | 場所の説明 |
|---|---|
| **Aショート** | Aサイトへの短い通路 |
| **Aロング** | Aサイトへの長い通路（シャワー側） |
| **シャワー** | Aロング途中の室内エリア |
| **Aヘブン** | Aサイト上のコンテナ上 |
| **Uホール（ランプ）** | AサイトからCTに繋がるU字型の通路 |
| **トラック** | Aサイト内のトラック（車両） |
| **テレポーター入口（A）** | AショートからBサイド方面へのテレポーター |

### Bサイト周辺
| コールアウト | 場所の説明 |
|---|---|
| **Bロング** | Bサイトへの長い通路 |
| **Bショート** | Bサイトへの短い通路（フーカ経由） |
| **フーカ** | Bショート手前の室内 |
| **エルボー** | Bロング途中のL字に曲がる地点 |
| **Bウィンドウ** | Bサイト内の窓がある建物 |
| **テレポーター出口（B）** | Bロング付近のテレポーター出口 |
| **バックサイト** | Bサイト奥の防衛寄りエリア |

## コールアウトを覚えるコツ

1. **カスタムマッチで散歩**: 一人でカスタムマッチに入り、ミニマップを見ながら各ポジションの名前を確認する
2. **プロの配信を見る**: VCで報告しているプロの配信を見ると、自然にコールアウトが耳に入ってくる
3. **1マップずつ覚える**: 全マップを一度に覚えようとせず、よくプレイするマップから1つずつ覚える
4. **実戦で使う**: 覚えたら積極的にVCで使う。間違えても問題ない──使わないと定着しない

---

コールアウトを覚えることは、VALORANTのランクを上げるための**最もローコストで効果の高い勉強法**だ。エイム練習に比べて地味だが、チームゲームである以上、正確な報告ができるプレイヤーは確実に味方から信頼される。
