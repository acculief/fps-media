**{{アストラ・ヤオ}}**はVer.2.0で実装されたS級エーテル属性・支援専門のエージェント。スターズ・オブ・リラ所属の天才ヴァイオリニストで、戦場では「アイドリック・カデンツァ」と呼ばれる演奏状態から味方を強化し続ける。

属性やチーム構成を問わず**あらゆるパーティに入る万能サポーター**として、実装以来"人権キャラ"の名をほしいままにしている。本記事では{{アストラ・ヤオ}}のビルドを徹底解説する。

## {{アストラ・ヤオ}}の基本情報

| 項目 | 詳細 |
|---|---|
| **レアリティ** | S級 |
| **属性** | エーテル |
| **専門** | 支援 |
| **所属** | スターズ・オブ・リラ |
| **キーステータス** | 攻撃力（ATK 3,429が目標ライン） |

{{アストラ・ヤオ}}のバフ量は**初期ATKに依存**する。コアパッシブ「アンダンテ・カンタービレ」によるATKバフは自身のATKの22〜35%（最大1,200Pt）。ATK 3,429に到達するとバフが上限に達するため、このラインを目指すのが最優先だ。

## コア性能の解説

{{アストラ・ヤオ}}の最大の特徴は**「アイドリック・カデンツァ」状態**だ。特殊スキル「風鈴といつかの約束」を使用すると演奏状態に入り、以下の効果が発動する。

- **チーム全体にダメージバフ**（最大+20%）と**会心ダメージバフ**（最大+25%）
- 被ダメージ40%軽減
- 味方がクイック支援を発動するたびに「ビブラート」（協力攻撃）が発生
- ビブラート発動時にコアパッシブのATKバフが味方全体に付与（20秒間）

つまり、**1回スキルを使ったらあとはフィールドに出る必要がない**。味方が戦っている間、バックグラウンドで延々とバフを撒き続けてくれる。

### スキル優先度

| 優先度 | スキル | 重要度 |
|---|---|---|
| **1位** | コアパッシブ「アンダンテ・カンタービレ」 | ★★★★★ |
| **2位** | 特殊スキル「風鈴といつかの約束」 | ★★★★★ |
| **3位** | 連携スキル「ほろ酔いコンツェルト」 | ★★★★ |
| **4位** | 支援スキル | ★★★ |
| **5位** | 通常攻撃「カプリチオ」 | ★★ |
| **6位** | 回避 | ★★ |

コアパッシブと特殊スキルが最優先。通常攻撃と回避はほぼ使わないため後回しでよい。

## おすすめ音動機（武器）

### Tier 1: モチーフ武器

**[[優美のヴァニティ]]**

- ATK 3,429ラインを達成しやすく、バフ量・エネルギー管理の両面で最適解

## おすすめドライバディスク

### 最優先セット: [[静寂のアストラ]]4 + [[スイング・ジャズ]]2

| セット | 効果 |
|---|---|
| **[[静寂のアストラ]] ×4** | 攻撃力+10%（2pc）。クイック支援で出場時に「天籟」を獲得し、与ダメージ+8%（最大3スタック＝+24%） |
| **[[スイング・ジャズ]] ×2** | エネルギー自動回復+20% |

名前の通り{{アストラ・ヤオ}}のために作られたセット。クイック支援のたびにダメージバフが積み重なり、最大3スタックで**チーム全体に+24%のダメージ上昇**。エネルギー管理を[[スイング・ジャズ]]2pcで補う。

### 代替セット

| セット構成 | 用途 |
|---|---|
| **[[静寂のアストラ]]4pc + [[ホルモン・パンク]]2pc** | モチーフ武器がない場合に。ATK+10%でATKライン到達を補助 |
| **[[月光騎士の讃歌]]4pc + [[スイング・ジャズ]]2pc** | 連携スキル重視の構成。[[静寂のアストラ]]の厳選が終わるまでの繋ぎとしても有効 |

### ディスクのメインステータス優先度

| スロット | 推奨メインステータス |
|---|---|
| **4番** | 攻撃力%（最優先） |
| **5番** | 攻撃力%（最優先） |
| **6番** | エネルギー自動回復（モチーフ武器あり） / 攻撃力%（モチーフ武器なし） |

### サブステータス優先度

**攻撃力% > 攻撃力実数 > エネルギー自動回復 > 会心ダメージ / 会心率**

とにかくATKを盛ることが最重要。ATK 3,429ラインを超えたら、エネルギー自動回復に比重を移してビブラートの回転率を高めよう。1凸所持時は目標ATKが約2,963まで下がるため、エネルギー自動回復を優先しやすくなる。

## おすすめチーム編成

{{アストラ・ヤオ}}は**どのチームにも入る万能サポーター**。ここでは特に相性の良い構成を紹介する。

### 編成1: 氷異常パーティ（環境最強クラス）

| 枠 | エージェント | 役割 |
|---|---|---|
| **メインDPS** | {{星見雅}} | 氷異常火力 |
| **サブDPS** | {{月城柳}} | 電気異常＋追撃 |
| **サポート** | {{アストラ・ヤオ}} | ATK・ダメバフ・会心ダメバフ |

{{星見雅}}と{{月城柳}}のダブルアタッカー構成。2人のキャラチェンジでクイック支援が頻発し、{{アストラ・ヤオ}}のビブラートとバフが途切れない。現環境トップの編成のひとつ。

### 編成2: 炎アタッカーパーティ

| 枠 | エージェント | 役割 |
|---|---|---|
| **メインDPS** | {{イヴリン}} | 炎属性メイン火力 |
| **撃破** | {{ライト}} | 撃破＋炎シナジー |
| **サポート** | {{アストラ・ヤオ}} | ATK・ダメバフ・会心ダメバフ |

{{イヴリン}}の高火力を{{アストラ・ヤオ}}のバフで最大化。{{ライト}}がスタン・撃破を担当し、{{アストラ・ヤオ}}のエネルギーが切れた際もクイック支援枠として機能する。

### 編成3: エーテルパーティ

| 枠 | エージェント | 役割 |
|---|---|---|
| **メインDPS** | {{朱鳶}} | エーテル属性メイン火力 |
| **撃破** | {{青衣}} | 電気スタン蓄積 |
| **サポート** | {{アストラ・ヤオ}} | ATK・ダメバフ・エーテルシナジー |

{{朱鳶}}のバレット管理と{{アストラ・ヤオ}}のクイック支援が噛み合う。キャラチェンジの多い{{朱鳶}}の立ち回りと相性抜群。

### 編成4: 汎用枠として

{{アストラ・ヤオ}}のバフは**属性を問わない**。物理・氷・炎・電気・エーテル──どのパーティでもサポート枠に入れるだけで火力が上がる。「サポート枠に迷ったら{{アストラ・ヤオ}}」が正解だ。

## 凸効果の評価

| 凸数 | 効果概要 | 優先度 |
|---|---|---|
| **0凸** | 基本性能で環境最強サポーター。無凸で十分実用的 | **推奨** |
| **1凸** | 敵の全属性耐性-6%（最大3スタック＝**-18%**）。終結スキルでデシベル1,000Pt即時獲得＋チーム無敵 | ★★★★★ |
| **2凸** | ATKバフ上限+400Pt（最大1,600Pt）。精密支援の追加発動 | ★★★★ |
| **4凸** | 終結スキル後15秒間、クイック支援のエネルギー消費なし | ★★★ |
| **6凸** | ビブラートとトーンクラスターのダメージ2倍、会心率+80% | ★★★ |

### 凸の推奨ライン

- **無凸**: サポーターとしての基本性能は無凸で完成。まずは無凸で運用を始めよう
- **1凸（最優先）**: **全属性耐性-18%**の恩恵が凄まじい。チーム全体のDPSが大幅に向上し、どのパーティでも恩恵がある。**モチーフ武器よりも1凸を優先すべき**というのが多くのプレイヤーの結論
- **2凸**: ATKバフ上限が上がり、ATKラインの要求値が約2,963まで下がる。ビルドの自由度が増す
- **6凸**: 火力面の恩恵は大きいが、サポーターとしてはオーバースペック。重課金向け

## 戦い方のコツ

1. **開幕即カデンツァ**: バトル開始直後に特殊スキル「風鈴といつかの約束」を使おう。初回は**エネルギー消費なし**で発動できる
2. **フィールドに出ない**: カデンツァ状態に入ったら、あとはメインDPSに切り替えるだけ。味方のクイック支援で自動的にビブラートが発動する
3. **ATK 3,429を目指す**: 無凸時のATKバフ上限に到達するライン。1凸なら約2,963でOK
4. **エネルギー管理**: ビブラート1回につきエネルギーを25消費。エネルギーが枯渇するとバフが途切れるため、[[スイング・ジャズ]]2pcやサブステのエネルギー自動回復で補おう
5. **終結スキルのタイミング**: HPが減った時や、バフを再度かけ直したい時に使用。チーム全体のHP回復＋次の2回のクイック支援が連携スキルに強化される
6. **1凸ならさらに強力**: 1凸の耐性ダウンはビブラートのたびに適用されるため、戦闘中は常時-18%が維持される

## まとめ

{{アストラ・ヤオ}}は**「どのパーティにも入る最強サポーター」**。属性やチーム構成を選ばず、カデンツァ状態で放置するだけでチーム全体の火力を大幅に引き上げてくれる。

育成の方針は**「とにかくATKを盛る」**の一言に尽きる。[[静寂のアストラ]]4セット＋ATK%メインステのディスクでATK 3,429ラインを目指そう。武器はA級の[[喧嘩腰のボンバルダム]]でも十分機能するため、余剰リソースがあれば**モチーフ武器よりも1凸を優先**するのがおすすめだ。

未所持のプレイヤーは復刻時に最優先で確保すべきエージェントのひとりだ。
