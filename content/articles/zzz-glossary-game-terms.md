ゼンレスゾーンゼロ（ゼンゼロ）にはHoYoverse独自の用語が多数登場する。本記事ではゲーム内で頻出する用語を**カテゴリ別**に整理して解説する。

---

## キャラクター関連

| 用語 | 読み方 | 説明 |
|---|---|---|
| **エージェント** | ─ | プレイアブルキャラクターのこと。S級・A級のレアリティがある |
| **プロキシー** | ─ | プレイヤーの分身。空洞案内人として活動する主人公兄妹 |
| **ボンプ** | ─ | 戦闘をサポートする小型ロボット。チームに1体編成できる |
| **アタッカー** | ─ | メインダメージを担うエージェントの役割 |
| **サポート** | ─ | バフや回復でチームを支援するエージェントの役割 |
| **スタン** | ─ | 敵のスタンゲージを蓄積してブレイク状態にする役割 |
| **異常（アノマリー）** | いじょう | 属性侵蝕を蓄積して爆発ダメージを与える役割 |
| **防御** | ぼうぎょ | シールドや被ダメ軽減でチームを守る役割 |

## 装備・育成

| 用語 | 読み方 | 説明 |
|---|---|---|
| **音動機** | おんどうき | エージェントが装備する武器。W-Engineとも呼ばれる |
| **ドライバディスク** | ─ | エージェントのステータスを強化する装備品。6枠セットで効果が発動 |
| **メインステータス** | ─ | ドライバディスクの主要ステータス。スロットごとに種類が決まる |

## ガチャ・通貨

| 用語 | 読み方 | 説明 |
|---|---|---|
| **ポリクローム** | ─ | ガチャ（シグナルサーチ）を回すための主要通貨 |
| **マスターテープ** | ─ | 限定エージェントガチャ用のチケット。ポリクローム160個で1枚 |
| **ボイルドウェーブテープ** | ─ | 音動機ガチャ用のチケット。ポリクローム160個で1枚 |
| **天井** | てんじょう | S級エージェント確定までの最大回数（90回） |
| **50/50** | フィフティ | 天井時にピックアップキャラを引く確率が50%であること |
| **すり抜け** | ─ | 50/50に負けてピックアップ以外のS級を引くこと |
| **確定枠** | かくていわく | すり抜け後の次の天井で100%ピックアップが出ること |

## コミュニティ用語

| 用語 | 説明 |
|---|---|
| **凸（とつ）** | 同じエージェントを複数回入手して性能を強化すること。0凸〜6凸まで |
| **無凸** | 凸をしていない状態（1体のみ所持）。多くのS級は無凸で十分強い |
| **厳選** | 理想のサブステータスを持つドライバディスクを繰り返し周回して集めること |
| **天井管理** | ガチャの天井カウントを把握して、計画的にポリクロームを使うこと |
| **餅武器** | モチーフ武器のこと。各エージェント専用の最適音動機 |
| **サブステ** | サブステータスの略称 |
| **DPS** | Damage Per Secondの略。単位時間あたりのダメージ量 |

## まとめ

ゼンゼロは独自の用語が多いが、基本的な用語を理解すればゲームの理解度が格段に上がる。特に**ドライバディスクの厳選**と**ガチャの天井管理**は、効率的にキャラクターを育成するうえで重要な概念だ。

わからない用語があったら、本ページをブックマークしていつでも確認してほしい。
