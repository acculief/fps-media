HoYoverseは2024年12月18日、アクションRPG『[ゼンレスゾーンゼロ](https://zenless.hoyoverse.com/ja-jp)』の大型アップデート**Ver.1.4「星流れ、神鳴の奔るが如く（Storm of Falling Stars）」**を配信開始した。

Ver.1.xシリーズ最後のアップデートとなる本バージョンでは、氷属性（霜烈）の最強異常アタッカー「{{星見雅}}（ミヤビ）」の実装と、無料配布エージェント「{{浅羽悠真}}」の追加が目玉。デシベル値の仕様変更など、ゲームシステムの大幅改修も行われた。

---

## 新エージェント「{{浅羽悠真}}（Asaba Harumasa）」──電気属性・攻撃のS級エージェント

Ver.1.4前半ガチャで実装される**{{浅羽悠真}}**は、新エリー都を拠点に活動するS級攻撃エージェント。双銃を操る射撃スタイルで、高速かつ華麗な連続攻撃を叩き込む。

![浅羽悠真（Asaba Harumasa）のキャラクターポートレート](/images/d5f38f82429a.png)

<small>※画像は『ゼンレスゾーンゼロ』公式より引用</small>

| 項目 | 詳細 |
|---|---|
| **名前** | {{浅羽悠真}}（Asaba Harumasa） |
| **レアリティ** | S級 |
| **属性** | 電気 |
| **専門** | 攻撃（アタック） |
| **所属** | セクション6 |

{{浅羽悠真}}の主な特徴は以下のとおり。

- **二丁拳銃による高速射撃**: 通常攻撃の連射速度が非常に速く、手数で圧倒するスタイル
- **リソース「弾倉」管理**: 固有リソース「弾倉」を消費してEXスキルを強化。弾倉がフル状態からの射撃は追加電気ダメージを発生させる
- **EXスキル「疾風弾幕」**: 大量の弾丸を浴びせる範囲攻撃。ブレイク状態の敵に対してはダメージ倍率が大幅に上昇
- **回避カウンター**: 完全回避成功時の反撃が非常に強力で、攻防一体の立ち回りが可能

電気属性パーティのメインアタッカーとして、既存の電気サポート・撃破エージェントとの組み合わせが有効だ。

## 新エージェント「{{星見雅}}（ミヤビ）」──霜烈属性・異常のS級エージェント

Ver.1.4のもう一人の新エージェント**{{星見雅}}（ミヤビ / Hoshimi Miyabi）**は、S級異常アタッカー。対ホロウ六課に所属する最年少の「虚狩り」であり、骸討ちの刀「無尾」を操る。

| 項目 | 詳細 |
|---|---|
| **名前** | {{星見雅}}（Hoshimi Miyabi） |
| **レアリティ** | S級 |
| **属性** | 霜烈（氷系特殊属性） |
| **専門** | 異常（アノマリー） |
| **所属** | 対ホロウ六課 |

- **霜烈属性の異常蓄積**: 敵を氷焔で包み込み、霜烈属性の状態異常で特殊効果を引き起こす
- **専用ゲージ「落霜」**: 「落霜」を消費して強力な広範囲攻撃を展開。霜灼・破や強化特殊で「落霜」をチャージ
- **自己完結型の高火力**: サポートに頼らずとも単体で凄まじい火力を発揮するハイパーキャリー型
- **環境を定義するエージェント**: 実装直後から式輿防衛戦で最強クラスの成績を叩き出し、氷属性パーティの核として不動の地位を確立

## メインストーリー追加──第5章

Ver.1.4では**メインストーリー第5章**が追加。対ホロウ六課を中心に物語が大きく展開し、新エリア「ポート・エルピス」が舞台となる。{{星見雅}}や{{浅羽悠真}}のバックストーリーも描かれ、ボイス付きのカットシーンも多数追加されている。

---

## デシベル値の仕様変更

Ver.1.4の大きなシステム変更として、**デシベル値**の仕組みが一新された。

- **変更前**: デシベル値はチーム全体で共有。誰か一人のアルティメットを使うと全員分が消費される
- **変更後**: デシベル値はキャラごとに個別で蓄積・消費する仕様に
- **影響**: 各キャラのアルティメットを個別に管理できるようになり、チーム運用の戦略性が大幅に向上

---

## 式輿防衛戦の更新

エンドコンテンツ**「式輿防衛戦」**が更新され、新たなステージと敵が追加された。

| 更新内容 | 詳細 |
|---|---|
| **新ステージ** | 電気・エーテル属性が有利なギミック構成 |
| **新ボスエネミー** | 高耐久の新エリートエネミーが登場 |
| **報酬更新** | クリア報酬にVer.1.4新素材が追加 |
| **難易度追加** | 上級者向けの高難易度ステージが新設 |

{{星見雅}}・{{浅羽悠真}}いずれも式輿防衛戦で活躍できるチューニングがされており、新キャラクターの性能を試す場としても最適だ。

---

## 新ボンプ「ニャーメリック」

Ver.1.4では新S級ボンプ**「ニャーメリック」**が追加。電気属性との相性に優れ、パーティに電気属性エージェントが2名以上いる場合に追加効果が発動する。{{浅羽悠真}}を軸とした電気パーティの編成に最適な選択肢となる。

## QoL改善まとめ

Ver.1.4では以下の利便性改善が実施された。

| 改善項目 | 内容 |
|---|---|
| **装備プリセット機能** | エージェントごとに装備セットを最大3つまで保存可能 |
| **自動周回機能拡張** | 素材ステージの自動周回回数が10回から20回に拡大 |
| **ボンプ管理改善** | ボンプの一覧画面にフィルター・ソート機能を追加 |
| **チャット機能強化** | フレンドとのチャットにスタンプ機能が追加 |
| **パフォーマンス改善** | ロード時間の短縮および一部端末でのメモリ使用量最適化 |
