Ver.2.6で零号ホロウに**「作戦行列・拡張モード」**が追加された。通常の作戦行列より難易度が大幅に上がっており、適切な準備なしでは返り討ちに遭う。

本記事では拡張モードの仕様と攻略法を解説する。

---

## 作戦行列・拡張モードとは

| 項目 | 詳細 |
|---|---|
| **開放条件** | 通常の作戦行列をクリア済み |
| **難易度** | 通常モードの約1.5〜2倍 |
| **ステージ数** | 全5ステージ（各ステージにボス戦あり） |
| **報酬** | ポリクローム、S級育成素材、限定称号 |
| **リセット** | バージョンごとにリセット |

通常の作戦行列との最大の違いは、**敵のHP・攻撃力の大幅上昇**に加え、各ステージに**特殊ルール（変則条件）**が付与される点。

---

## 各ステージの特殊ルールと対策

### ステージ1: 電気耐性強化

**特殊ルール**: 敵の電気耐性が50%アップ

**対策**:
- 電気アタッカー（{{月城柳}}（ヤナギ）など）は火力が大幅に落ちるため、**物理or炎アタッカー**に切り替えるのが無難
- {{月城柳}}（ヤナギ）の電気異常蓄積も効率が悪くなるため、このステージでは他属性を推奨
- おすすめ: {{バーニス}}（炎）、{{朱鳶}}（エーテル）

### ステージ2: 回避制限

**特殊ルール**: 回避のクールダウンが2倍に

**対策**:
- パリィ（受け流し）の重要性が跳ね上がる
- **シールド持ちサポート（{{セス}}等）**を編成して被ダメを軽減
- 無理に回避せず、距離を取ってからスキルで攻撃するヒット＆アウェイ戦法が有効

### ステージ3: 時間制限

**特殊ルール**: 各戦闘に3分の制限時間

**対策**:
- DPS最優先の編成が必須。サポート過多は避ける
- **スタン→連携スキル→大ダメージ**のサイクルを高速で回す
- おすすめボンプ: バトラー（連携スキル威力バフ）

### ステージ4: 侵蝕フィールド

**特殊ルール**: フィールドに持続ダメージ地帯が出現

**対策**:
- **回復持ちエージェント**の編成が必須（{{ルーシー}}、{{ニコ}}等）
- ダメージ地帯を避けながら戦うため、機動力の高いキャラが有利
- レゾブレムで回復系を優先的にピック

### ステージ5: 全ルール複合（最終ステージ）

**特殊ルール**: ステージ1〜4のルールがランダムで2つ同時適用

**対策**:
- どの組み合わせが来ても対応できる**バランス型チーム**を用意
- 推奨: 汎用DPS＋シールドサポート＋スタン役
- レゾブレムの選択が極めて重要。後述の「おすすめレゾブレム」を参照

## おすすめレゾブレム構成

### 攻撃型（ステージ3向け）

| レゾブレム | 効果 | 優先度 |
|---|---|---|
| **臨界の鋭刃** | 会心率+15%、会心ダメージ+30% | ★★★★★ |
| **属性増幅** | 該当属性ダメージ+25% | ★★★★ |
| **連撃の波動** | 通常攻撃ダメージ+40% | ★★★★ |
| **破砕の一撃** | スタン中の敵へダメージ+35% | ★★★ |

### 耐久型（ステージ4向け）

| レゾブレム | 効果 | 優先度 |
|---|---|---|
| **生命の息吹** | HP30%以下で自動回復（戦闘中1回） | ★★★★★ |
| **鉄壁の守護** | 被ダメ-15% | ★★★★ |
| **回復の波動** | スキル使用時HP5%回復 | ★★★ |

### 汎用型（ステージ5向け）

攻撃型と耐久型をバランスよくピック。最低でも「生命の息吹」は確保しておきたい。

## 効率周回のコツ

1. **レゾブレムの取捨選択が最重要**: 不要なレゾブレムは即スキップして先に進む
2. **ショップではHP回復アイテムを優先購入**: 特にステージ4以降で生存力が問われる
3. **カードの選択**: 攻撃系カードは序盤に、防御系カードは後半に取るのが効率的
4. **失敗したらチーム編成から見直す**: レゾブレム運より編成のほうが結果に影響する

## まとめ

作戦行列・拡張モードはゼンレスゾーンゼロの中でもトップクラスの難易度だが、報酬のポリクローム700個は非常に美味い。

攻略のカギは**特殊ルールに合わせたチーム切り替え**と**レゾブレムの取捨選択**。全ステージを無理に同じチームで突破しようとせず、ステージごとに最適な編成を考えよう。
